Contains all three samples for Chapter 1. 

* rot13.py - The first version of the ROT13 encoder.
* rot13-b.py - The second version, adding support for standard in.
* rot13-c.py - The third version, letting us toggle tag processing on and off.
* sample_page.html - The example HTML document used in the chapter.

* Python3 - Python 3 Code for the examples.

This code should run unmodified using Python 2.4 or above. There are 
no third party modules or extra configuration steps required.
