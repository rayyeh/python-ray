Contains all of the code samples from Chapter 9.


* Workbook1.csv - CSV files used during the Excel examples.
* __main__.odt - Output of Open Office/Introspection examples.
* intro.txt - Introductory text used during PDF generation.
* odt_build.py - Code used to generate an ODT file.
* output.pdf - Output of the PDF generation examples.
* paragraphs.txt - Paragraph information used during PDF generation.
* pdf_build.py - Code used to build PDF.
* profit.xls - Excel output file.
* xls_build.py - Code used to build an Excel sheet.

* Python3 - Python 3 Code for the examples. 

This code should run unmodified using Python 2.6 or above. These examples are not available for Python 3.
