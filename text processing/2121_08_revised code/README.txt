Contains all of the code samples from Chapter 8.


* lang - Language files needed to run the translated application.
* russian.txt - Small sample UTF-8 file contianing non-ASCII Chars.
* utf_coding.py - Counts bytes and compares to characters.
* utf_copy-a.py - Updates to copy per char instead of per byte.
* utf_copy-b.py - i18n version of utf_copy.
* utf_copy.pot - POT File for translations.
* utf_copy.py - UTF Copy file that causes an exception due to out of range char.
* utf_translate.py - Translates between encoding types.

* Python3 - Python 3 Code for the examples. 

This code should run unmodified using Python 2.6 or above.
