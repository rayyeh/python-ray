Contains all of the code samples from Chapter 4.

* Workbook1.csv, Workbook1.xls, Workbook2.csv, Workbook2.xls, Workbook2a.csv - Various
  Excel & CSV data used during the csv module examples.
* csv_reader-b.py - The "formulas don't work" example.
* csv_reader-c.py - First non-Excel example, using a Unix password file.
* csv_reader-d.py - Translating CSV information into JSON output.
* csv_reader.py - Our first CSV example.
* csv_translate.py - Create a spreadsheet of Unix users (password format into
  CSV.
* default_args.py - Our defaults example, overwriting configuration options.
* example3.log - Raw log data.
* logscan-h.py - Version of log scanner using an INI-like configuration.
* logscan-i.py - Version of our log scanner handling default options.
* logscan.cfg - Configuration example for use with logscan.py & friends.
* passwd - An example Unix password file.
* passwd.csv - Unix password in CSV format.
* text_beginner - A package structure compatable with SetupTools.
* users.json - The output generated by our JSON example, suitable for importing
  back in.

* Python3 - Python 3 Code for the examples. This code has been tested to work with 
  VirtualEnv3 & distribute, under Python 3.1. The 'python setup.py develop' may
  require a --prefix option into the root of the Python 3 virtualenv.

This code should run unmodified using Python 2.4 or above. If you are using a version
lower than 2.6, you'll need the third party simplejson package.
