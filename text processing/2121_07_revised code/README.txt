Contains all of the code samples from Chapter 7.



* header.html - A simple header included by index.html
* index.html - A very basic Mako example
* render_mail.py - The driver program for all of our Mako examples found in the templates directory.
* page.html - A simple HTML template using 'self' syntax.
* rendered_page.html - A rendered version of page.html.
* templates - The collection of Mako templates that we use throughout the chapter.

* Python3 - Python 3 Code for the examples. 

This code should run unmodified using Python 2.4 or above.
