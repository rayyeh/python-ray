Contains all of the code samples from Chapter 5.

* text_beginner - The packages installed in this chapter. Specifically, the MX record ordering application and supporting files.
* url_regex.py - URL Validation Script.

* Python3 - Python 3 Code for the examples. This code has been tested to work with 
  VirtualEnv3 & distribute, under Python 3.1. The 'python setup.py develop' may
  require a --prefix option into the root of the Python 3 virtualenv.

This code should run unmodified using Python 2.4 or above.
