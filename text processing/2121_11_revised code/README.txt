Contains all of the code samples from Chapter 11.

* about_python.odt - Our example ODT file, indexed.
* c.l.py - The comp.lang.python sources.
* clp_index.py - Create a Nucular Index.
* clp_index-a.py - Update nucular index to include field-based data.
* clp_index-b.py - ODF Indexing.
* field_search.py - Searching based on field values.
* mcneil.xml - XML Search definition.
* search_for.py - Nucular queries with Python.
* text_scan.py - Linear search code used at the beginning. 

* Python3 - Python 3 Code for the examples. 

This code should run unmodified using Python 2.6 or above. The nucular examples are not 
compatible with Python 3.
