Contains all of the code samples form chapter 2

* example.log - Raw log data.
* example-b.log - Preformatted Log data.
* example.log.gz - Compressed log data.
* html_spelling.py - The initial local version of the spelling application.
* html_spelling-b.py - Updated version to use urllib2 module.
* html_spelling-c.py - Extended Error Handling
* index.html - The main page used with our local spelling application.
* air_speed.html - The linked page used in our spelling application.
* logscan.py - Our original logscan.py application, for parsing Apache logs.
* logscan-b.py - Introduce different types of source file parsing.
* logscan-c.py - Update to support direct file IO.
* logscan-d.py - Managing compressed files.

* Python3 - Python 3 Code for the examples.

This code should run unmodified using Python 2.4 or above. There are some third party
library requirements for the spelling examples. Note that the spelling examples are not
compatable with Python 3.
